1. Clone

Permite clonar un repositorio remoto a un workspace.

git clone [path]

Ejemplo:

git clone https://github.com/jhonnyesquivel/git-basics.git

3. Add

Indica a la terminal los cambios que seran incluidos en el siguiente commit.

git add <.|filename>

Si se agrega "." como filename, se agregaran todos los archivos modificados

Si solo se desea agregar un archivo, basta con escribir el nombre del mismo luego del comando.

Ejemplo:

git add .
git add index.html
