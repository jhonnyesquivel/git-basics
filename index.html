1. Clone

Permite clonar un repositorio remoto a un workspace.

git clone [path]

Ejemplo:

git clone https://github.com/jhonnyesquivel/git-basics.git

3. Add

Indica a la terminal los cambios que seran incluidos en el siguiente commit.

git add <.|filename>

Si se agrega "." como filename, se agregaran todos los archivos modificados

Si solo se desea agregar un archivo, basta con escribir el nombre del mismo luego del comando.

Ejemplo:

git add .
git add index.html

4. Commit

Registra todos los cambios pendientes el en repositorio local y les asigna un commit-id

git commit -m "mensaje descriptivo de los cambios"

Ejemplo:

git commit -m "se agrega el archivo index al repositorio"

6. Push

Sube un los cambios de un branch a un repositorio remoto. Se debe especificar el origin si el branch no existe remotamente.

git push --set-upstream origin <branch> git push

Ejemplo:

git push --set-upstream origin feature/pull
git push

8. Rebase

Aplica los commits de un branch en en el top de un branch especificádo. Requerirá ejecutar el comando: git push --force

git rebase <branch>

Ejemplo:

git rebase feature/pull
